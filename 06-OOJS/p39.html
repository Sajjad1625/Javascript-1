JavaScript Encspsulation

It is process of binding the data with the functions acting on that data. it allows us to control the data and validate it.

Points to remember..!
  • Use let and const keyword to make data mambers private.
  • Use setter method to set the data and getter mathod to get that data

Encspsulation allows us to handle an object using the following properties.

Read/Write - Here, we use setter method to write the data and getter method to read that data.

Read Only - In this case, we use getter methods only.

Write Only - In this case, we use setter methods only.
